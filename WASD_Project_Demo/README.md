### WSAD project

동서남북 각 위치에서 살고있는 WASD의 모험


<hr>

* **2D 플랫포머 게임**
  * 구현 기능
    - 플레이어 움직임
    - 몬스터 움직임
    - 2D 플랫포머
    - 장애물 패턴
    - 아이템
    - 캐릭별 씬 전환
    - 플레이어 공격 모드
    - 보스 패턴 - 각도 계산 등 (예정)
    - 맵구현

사용 언어 및 도구 : C# / Unity엔진 / PixelArt
